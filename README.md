# GarmentshopPOS 🧵

A complete Point of Sale (POS) system built for garments and fabric stores. 
Developed using C# Windows Forms (.NET Framework 4.8) with SQL Server Express as the backend database.

## Features

- 🔐 Role-based authentication (Admin / Employee)
- 👔 Separate sections for Gents and Ladies fabric management
- 📦 Dynamic stock management with barcode scanner support
- 💰 Complete sales entry with live receipt preview
- 🧾 Receipt generation and printing
- 📊 Audit & performance dashboard
- 📈 Stock review with date-range filtering
- ⚠️ Real-time low stock alerts
- 💳 Multiple payment methods (Cash, Card, EasyPaisa, JazzCash, Credit)
- 🪡 Ladies-specific fields (Print type, Embroidery, Suit type)
- 👥 User management (Admin only)
- 💾 SQL Server Express local database

## Tech Stack

- Language: C# (.NET Framework 4.8)
- UI: Windows Forms (WinForms)
- Database: SQL Server Express
- IDE: Visual Studio 2022

## Setup

1. Clone the repository
2. Install SQL Server Express
3. Run the SQL script in `/Database/schema.sql`
4. Update the connection string in `App.config`
5. Build and run in Visual Studio
